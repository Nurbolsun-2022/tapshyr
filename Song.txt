Керим Турапов
Туулган жерди сагынуу
                Кылымдардан кылым карытып,
                Кызыл-Суунун суусу агылып.
                Бал жыттанган таттуу кымыздай,
                Туулган жердин суусун сагындым.
                Жүрсөм дагы алыс араңдан,
                "Ата Журтум кетпейт санаамдан.\n" +
                "Акыл айтып, асыл кеп айткан,\n" +
                "Аталардын кебин сагынам.\n" +
                "Сары-Ой менен Көк-Ой тоосунан,\n" +
                "Бүркүт шаңшып, күкүк сайраган.\n" +
                "Ышкын терип аска-зоосунан,\n" +
                "Жүргөн күнүм эстеп сагынам.\n" +
                "Жылаңайлак, чырпык ат минип,\n" +
                "Жылгалардан тунук суу кечип.\n" +
                "Балтырымды тилип караган,\n" +
                "Балалыгым эстеп сагынам.\n" +
                "Коңур-Дөбө — жайлоо, жашыл төр,\n" +
                "Коңур желдүү, салкын, асыл жер.\n" +
                "Кокту, белес, күнгөй-тескейден,\n" +
                "Кой кайтарган күнүм сагынам.\n" +
                "Түндүгүнөн түтүн булатып,\n" +
                "Апам жапкан нанды сагынам.\n" +
                "Үйүр-үйүр жылкы айдаган,\n" +
                "Атам жүргөн жерди сагынам.\n" +
                "Үйүр-үйүр жылкы айдаган,\n" +
                "Атам конгон журтту сагынам.
